MongoDB::prevError

(PECL mongo >=0.9.5)

MongoDB::prevError — データベース操作中の直近に発生したエラーを調べる

説明

public array MongoDB::prevError ( void )

パラメータ

この関数にはパラメータはありません。

返り値

エラーと、それが何回前の操作で発生したかを返します。
