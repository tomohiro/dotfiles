ArrayAccess::offsetGet

(PHP 5 >= 5.1.0)

ArrayAccess::offsetGet — オフセットを取得する

説明

abstract public mixed ArrayAccess::offsetGet ( mixed $offset )

指定したオフセットの値を返します。

このメソッドは、オフセットが empty() かどうかを調べる際に実行されます。

パラメータ

offset

取得したいオフセット。

返り値

すべての型の値を返すことができます。
